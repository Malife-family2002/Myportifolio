Project Overview

This project showcases the practical application of web development skills acquired during training at UDOM. The goal was to build a fully functional portfolio website using modern technologies and best practices.

Features

  • Semantic HTML5 structure
  • CSS3 styling and responsive layout
  • JavaScript interactivity and DOM manipulation
  • Form validation with regular expressions

Technical Stack

This project was built using the following technologies:

  • HTML5
  • CSS3 (external, internal, and inline)
  • JavaScript ES6+

Outcome

The website effectively demonstrates design and development proficiency and serves as a live resume for job applications and freelance opportunities.
